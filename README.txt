Publishing:
1. Create a GitHub repository, for example: nosmo-website
2. Upload all files and folders exactly as they are.
3. Go to Settings -> Pages.
4. Choose Deploy from branch -> main -> /root.
5. Save.
6. Later connect your purchased domain.

The robot images are described as concept visuals only, not finished commercial products.
